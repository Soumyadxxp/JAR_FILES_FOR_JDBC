# JAR_FILES_FOR_JDBC

This repository contains the JDBC driver JAR files required to connect Java applications with different databases.

These JAR files are useful for Java SE projects, Java Swing applications, NetBeans projects, Eclipse projects, and other IDEs where database connectivity is required.

---

## Contents

### MySQL JDBC Connector

Contains the JDBC driver required to connect Java applications with MySQL Database.

---

### Oracle 10g JDBC Driver

Contains the JDBC driver for Oracle Database 10g.

---

### Oracle 11g JDBC Driver

Contains the JDBC driver for Oracle Database 11g.

---

### Python Oracle

Contains files required to establish Oracle Database connectivity using Python.

---

## How to Use

### NetBeans

1. Right-click your project.
2. Select **Properties**.
3. Open **Libraries**.
4. Click **Add JAR/Folder**.
5. Select the required JDBC driver.

### Eclipse

1. Right-click the project.
2. Select **Build Path**.
3. Choose **Configure Build Path**.
4. Click **Add External JARs**.
5. Select the required JDBC driver.

### IntelliJ IDEA

1. Open **Project Structure**.
2. Navigate to **Libraries**.
3. Click **+**.
4. Select the required JAR file.
